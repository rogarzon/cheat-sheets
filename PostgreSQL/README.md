# Commands

* Show roles: `\du`
* Show tables: `\dt`
* Show databases: `\l`
* Connect to a database: `\c <database>`
* Show columns of a table: `\d <table> or \d+ <table>`
* Quit: `\q`


# Cambiar a la cuenta de postgres
